Projects

UBC Data Science Project

As part of a data science project at UBC, we worked with data from PLAIcraft — a Minecraft server run by the Pacific Laboratory for Artificial Intelligence (PLAI) — to help the research group better understand which types of players are likely to contribute the most gameplay data. Specifically, we examined how player age might relate to total playtime. Using R in a Jupyter Notebook environment, we performed exploratory data analysis with the tidyverse package, built and evaluated a linear regression model using tidymodels, and assessed model performance through diagnostic plots. Our goal was to help PLAI target their recruitment efforts and manage server resources more effectively.

Tech used: R, Jupyter Notebook

Packages used: tidyverse, tidymodels and repr

The Budget Manager

The Budget Manager is a Java-based personal finance tool developed as part of a course project at the University of British Columbia between January and April 2025. Designed specifically for students transitioning into adulthood, the application features a clean and intuitive Java Swing user interface that allows users to easily log income, expenses, and savings. Data is persistently stored using JSON file storage, enabling easy access and reliable backups. With over 95% test coverage using JUnit, the application is built to be both robust and reliable. Whether it's setting savings goals or tracking daily spending, the Budget Manager empowers students to develop smart financial habits with confidence.

Tech used: Java, Swing, JUnit, JSON
